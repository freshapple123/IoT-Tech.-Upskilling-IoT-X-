# IoT_ai_Make-A-Thon
IoT Tech. Upskilling” 현장문제 중심 IoT-X 융합 메이커톤

# 아두이노 클라우드를 활용한 스마트 작업대 조명 시스템

## 기능

### 실시간 관측
- **온도, 습도, 조도**를 실시간으로 관측 가능

### 모드
- **여러가지 모드** 지원
- **블루투스 모드** 지원
- **음성 AI 인식 모드** 지원

### AI 손 트래킹 모드
- **AI 손 트래킹 모드** 설정 시:
  - 카메라로 손을 인식하여 조명장치가 손의 위치를 추적
  - 넓은 작업대 환경에서 조명장치 한 개로 작업 위치를 비추어 효율성 증가
